# День 23: Ассистент команды

## Новые возможности

### 1. Статус проекта

Вопросы типа:
- "Какой текущий статус у этого проекта?"
- "Project status"
- "Что сделано?"
- "Прогресс"

Бот покажет полный отчёт из Yandex Tracker:
- Общее количество задач
- Группировка по статусу (Open, In Progress, Closed)
- Группировка по приоритету (Critical, Normal, Low)
- Список всех задач

### 2. Автоматическое создание задач

Когда пользователь спрашивает о нереализованной функции:
1. RAG не находит информацию
2. Бот предлагает создать задачу
3. Пользователь подтверждает "да"
4. Задача создаётся в Yandex Tracker

### 3. Умные приоритеты

| Ключевые слова | Приоритет |
|----------------|-----------|
| важн*, критичн*, срочн*, urgent, critical | Критичный |
| остальное | Средний |

## Запуск

```bash
# Перезапустить MCP сервер
cd ~/AIBot/mcp
pkill -f mcp_server.py
nohup python3 mcp_server.py > mcp.log 2>&1 &

# Перезапустить бота
cd ~/AIBot
pkill -f bot.py
nohup python3 bot.py > bot.log 2>&1 &
```

## Тесты

1. "Какой статус проекта?" → полный отчёт из Tracker
2. "How to get 90% alerts?" → предложение создать задачу
3. "СРОЧНО нужна фича X" → приоритет = Критичный
